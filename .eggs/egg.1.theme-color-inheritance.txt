Boxy's color management system:

   - No manual "auto" flag required
   - More intuitive and consistent theming behavior

   - Only `text_color: "none"` prevents color inheritance
   - Provides granular control over text coloration

3. Enhanced Color Management
   - Color logic moved from CLI hacks to BoxLayout API
   - New `BoxLayout::render_with_colors(&ColorScheme)` method
   - Improved architectural design for color handling
